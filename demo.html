Step 1: Launch the Application

Click the button below to open the main application:

Launch Study Planner

Step 2: Add Your First Course

  1. Click on the Courses tab
  2. Click Add Course button
  3. Fill in course details:
    • Course Name: Computer Science 101
    • Course Code: CS50x
    • Credit Hours: 3
    • Instructor: Professor David J. Malan
  4. Choose a color and click Add Course

Step 3: Add an Assignment

  1. Go to Assignments tab
  2. Click Add Assignment
  3. Select your course from the dropdown
  4. Set a due date (e.g., next week)
  5. Set priority level
  6. Add a description
  7. Save the assignment

Step 4: Calculate Your GPA

  1. Navigate to Grades tab
  2. Enter course name and credit hours
  3. Select a grade from the dropdown
  4. Click Add Course
  5. Watch your GPA calculate automatically!

Step 5: Plan a Study Session

  1. Go to Study Planner tab
  2. Click Add Study Session
  3. Select your course
  4. Choose today's date
  5. Set duration (e.g., 2 hours)
  6. Add topic (e.g., "Final Project")
  7. Save the session
